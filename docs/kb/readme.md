首先，到Ollama官网下载Ollama客户端
安装完成后，在开始界面输入框右下角的模型列表中选择适合的模型（我使用qwen3.5 9B）
选择完毕后，在输入框中输入内容，就会自动开始下载模型
下载完毕后，再到Ollama官网，点击左上角 Models，选择Embedding，下载qwen3-embedding或者bge-m3，方法为点击进去复制CLI到cmd使用
其次，到Anything LLM官网下载Anything LLM客户端
安装完成后，设置大模型的服务提供商为Ollama，再从模型中选择刚才下载的本地大模型，然后点击右侧箭头进行下一步
进入工作区后，点击左下角设置，在嵌入模型处选择刚才下载的本地嵌入模型，然后在文本分割中调整切片长度为1000，上下文保留200
在左侧工作区可以上传文档，移动到工作区后选择嵌入
